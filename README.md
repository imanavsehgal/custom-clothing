# Custom Clothing - Next.js Starter

This repo contains a minimal Next.js + Tailwind project with a React-Konva customizer demo.

## Quick start (in a Codespace or cloud workspace)

1. `npm install`
2. `npm run dev`
3. Open http://localhost:3000

## Notes
- Customizer is at `/customize`.
- Install deps: `react-konva` + `konva`.
- Tailwind is configured; edit styles/globals.css to change styling.
